Customercare Agent

An AI-powered customer care and support analytics system built using Agentic AI workflows, SQL, and structured customer support data.
This project demonstrates how Large Language Models (LLMs) can interact with a relational customer support database to answer operational, analytical, and customer-service-related questions in natural language.
---

## Features

- AI-powered customer support assistant
- Natural language to SQL querying
- Customer care analytics
- Product issue tracking
- Ticket lifecycle management
- Resolution performance tracking
- Multi-level support agents (L1, L2, L3, Escalation, QA, Management)
- Electronics product catalog with 50+ products
- Support ticket categorization and SLA tracking
- Resolution and refund management
- Dashboard-ready SQL views
- Docker-compatible database initialization
